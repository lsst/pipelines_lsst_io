This is a minor feature release based on releases :ref:`release-v29-0-0` and :ref:`release-v29-1-0` with 2 additional tickets back ported.
This release fixes python memory leaks (see :jira:`DM-38772`) by using holder classes introduced in pybind11 version 3 and therefore is not ABI compatible with older releases.
For a complete list of changes made, see :doc:`tickets/v29_2_0`.

The `Characterization Metric Report (DMTR-461) <https://dmtr-461.lsst.io/>`_ describes the scientific performance of this release in terms of scientific performance metrics.
